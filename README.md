Este projeto é uma página web interativa que apresenta uma linha do tempo sobre a história da Capoeira, desde suas raízes no Brasil Colonial até seu reconhecimento como Patrimônio Cultural da Humanidade.
O objetivo é oferecer uma visualização clara e informativa dos principais marcos históricos da Capoeira, utilizando dados carregados dinamicamente.

 - A aplicação foi desenvolvida como parte da Imersão em Desenvolvimento da Alura, focando na manipulação do DOM com JavaScript e no consumo de dados a partir de um arquivo JSON.

Funcionalidades
  - Linha do Tempo Dinâmica: Os eventos históricos são carregados a partir de um arquivo data.json e renderizados como cards na página.
  - Busca Interativa: Um campo de busca permite filtrar os eventos da linha do tempo em tempo real, facilitando a navegação por períodos ou termos específicos.
  - Links Externos: Cada card contém um link "Saiba mais" para aprofundar o conhecimento sobre o evento correspondente.
  - Design Responsivo: A interface foi estilizada para se adaptar a diferentes tamanhos de tela.
Tecnologias Utilizadas
  - HTML5: Para a estrutura semântica da página.
  - CSS3: Para a estilização dos componentes, layout e design.
  - JavaScript: Para a lógica de programação, manipulação, carregamento dos dados do JSON e implementação da funcionalidade de busca.
  - JSON: Como formato para armazenar e fornecer os dados da linha do tempo de forma estruturada.
